## 3. Системные состояния

- [ ] Меню (closed/open/focus/active).
- [ ] Модалки (default/loading/error/success).
- [ ] Уведомления (info/success/warning/error).
- [ ] Page states: `empty/loading/error/forbidden`.

## 4. Токены и styleguide

- [ ] Typography tokens.
- [ ] Color tokens.
- [ ] Spacing/radius/shadow tokens.
- [ ] Semantic status tokens.
- [ ] Input states.

- [ ] Motion tokens (duration/easing).

## 5. Atomic mapping

- [ ] Atoms размечены.
- [ ] Molecules размечены.
- [ ] Organisms размечены.
- [ ] Template уровень выделен.
- [ ] Page-уровень собран.

## 6. UX/UI premium quality

- [ ] Один главный визуальный фокус на экран.
- [ ] `Primary CTA` консистентен с метаданными wireframe.
- [ ] Progressive disclosure реализован без перегрузки интерфейса.
- [ ] A11y-основания проверены (контраст/читаемость/focus path).
- [ ] Trust/privacy риски учтены в чувствительных сценариях.

## 7. Motion и a11y

- [ ] Motion имеет функциональную цель.
- [ ] Нет избыточных декоративных анимаций.
- [ ] Описан `prefers-reduced-motion` fallback.
- [ ] Анимации не создают layout-рисков.

## 8. Передача в разработку

- [ ] Нейминг узлов консистентен.
- [ ] Нет неоднозначных состояний.
- [ ] Макет готов к 1:1 реализации.
- [ ] Есть mapping `wireframe -> .pen -> components`.
